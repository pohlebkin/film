
# get-list-years - получить список годов выпусков фильмов

## Пример запроса

```
https://film-backend.komaf.ru/api/get-list-years
```

## Пример ответа

```
{
    "status": "ok",
    "list": {
        "2021": {
            "id": "2021",
            "count_films": "1"
        },
        "2022": {
            "id": "2022",
            "count_films": "1"
        }
    }
}
```

## Примечание

```id``` годов совпадают собственно с самими годами
